Dynamic Table Angular Component

An example of developing a reusable component in Angular that does these things:

  • Generates dynamic table headers (column names) based on data feeds
  • Generates dynamic filters (facets) based on data feeds
  • Reports front-end errors to a hosted service on same or different domain (Airbrake.io).
These alerts can be thrown programmatically or manually as shown here, and they are all logged to Airbrake.io.
